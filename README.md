# Text Generation with GPT-2

## Objective
To generate coherent and contextually relevant text using the GPT-2 transformer model.

## Technologies Used
- Python
- Google Colab
- Hugging Face Transformers
- PyTorch

## Features
- Loads pretrained GPT-2 model
- Fine-tunes on custom dataset
- Generates AI-based text from prompts
